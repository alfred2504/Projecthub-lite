# ProjectHub Lite

Frontend MVP - Next.js + TypeScript + Tailwind + Zustand

## Run locally
1. `npm install`
2. `npm run dev`
3. Open http://localhost:3000

## Features
- Create / Update / Delete tasks
- Mark complete / undo
- Filter by priority & status
- LocalStorage persistence
- Light / Dark mode toggle

## Notes
- App Router (src/app) required
- `next.config.ts` experimental.appDir must be true
